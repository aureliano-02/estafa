Cuando se utiliza un explorador como Google, Chrome, Firefox, opera, iexplorer, etc, es obligatorio registrarse con clave y usuario.

Peor aun: si intenta otro usuario, el sistema le pedira ingresar de nuevo.

El perfil es unico. Esto es valido no solo para WS, Email, telegram etc, es valido aun para el simple uso de un explorado

Muchas veces es utilizado de forma automatica, esto solo indica que quien accedio por primer vez lo dejo abierto.
